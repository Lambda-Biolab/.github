# Lambda Biolab

Basel's independent DIY molecular biology lab.

Benches, reagents, and compute for scientists, students, and tinkerers at the intersection of biology, computation, and hardware.

**[lambconsulting.bio/lambda-biolab](https://www.lambconsulting.bio/lambda-biolab)** · Riehenstrasse 14, 4058 Basel · BSL-1

---

## What we run

- **Wet lab** — BSL-1 bench space, cloning reagents, cell lines, eLabFTW notebooks, instruments.
- **Dry lab** — on-site workstation (i9 + RTX 4090), 3D printers, soldering, wiring, building.
- **Community** — hackathons, workshops, open build nights.
