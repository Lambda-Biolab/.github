# Lambda Biolab                                                                                                                                                                                     
Basel's independent DIY molecular biology lab. 

Benches, reagents, and compute for scientists, students, and tinkerers working at the intersection of biology, computation, and hardware.   

**[lambdabiolab.com](https://lambdabiolab.com)** · Riehenstrasse 14, 4058 Basel · BSL-1                                                                                                             
                                                                          
  ---                                                                                                                                                                                                                                                                                                                                                                                             
  ### What we run                                                                                                                                                                                     
- **Wet lab** — BSL-1 bench space, cloning reagents, cell lines, eLabFTW notebooks, shared instruments.                                  
- **Dry lab** — on-site workstation (i9 + RTX 4090), 3D printers, and space to solder, wire, and build.
- **Community events** — hackathons, workshops, and open build nights.                                                                                                                              
### Active projects                                                                                                                                                                                                     
  - [**lambda-biolab-hack-a-thon**](https://github.com/Lambda-Biolab/lambda-biolab-hack-a-thon) — site for **Lambda Hack Basel** (May 16, 2026). Five tracks, one day, open source required.                                                                                                                                                                             
- [**lambda-biolab**](https://github.com/Lambda-Biolab/lambda-biolab) — eLabFTW + OpenCloning Docker stack powering the lab.
