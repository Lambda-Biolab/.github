# Contributing

Lambda Biolab ships in public. Thanks for wanting to help.

## Issues

Open issues in the repo they concern, not here. Include what you observed, what you expected, and how to reproduce.

## Pull requests

- One topic per PR. Small and focused over complete.
- Describe what changed, why, and how you tested.
- Dev setup lives in each repo's README.
- No CLA. By opening a PR you agree your contribution is licensed under that repo's LICENSE.

## Code of conduct

This project follows [`CODE_OF_CONDUCT.md`](./CODE_OF_CONDUCT.md). Be kind.

## Questions

Open a [discussion](https://github.com/orgs/Lambda-Biolab/discussions) or reach out via [lambconsulting.bio/lambda-biolab](https://www.lambconsulting.bio/lambda-biolab).
